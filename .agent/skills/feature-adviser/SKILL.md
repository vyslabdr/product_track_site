---
name: Feature Adviser
description: Analyze project changes and suggest feature improvements
---

1. Scan project changes
   - Detect newly added or modified components, pages, or modules.

2. Analyze impact
   - Check dependencies, performance impact, and code complexity.
   - Detect potential missing features or enhancements.

3. Suggest improvements
   - Provide prioritized list of recommended features or optimizations.
   - Suggest refactors for better maintainability.

4. Generate report
   - Create a concise report with:
       - Feature suggestions
       - Potential risks
       - Estimated effort

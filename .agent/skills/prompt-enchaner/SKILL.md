---
name: Prompt Enhancer
description: Enhance AI prompts for better output and accuracy
---

1. Receive prompt
   - Take a raw or user-provided prompt as input.

2. Analyze intent
   - Detect goal, scope, and context of the prompt.
   - Identify missing details or ambiguities.

3. Improve prompt
   - Rewrite prompt to be clearer, more precise, and actionable.
   - Suggest optional constraints, formatting, or examples.

4. Output enhanced prompt
   - Provide final enhanced prompt ready for use.
   - Include summary of changes and suggestions applied.
